NomenclatureJS is a simple tool to facilitate the writing of coding conventions.

Coding conventions are a set of rules for a specific programming language that recommend programming styles, best practices, and methods for maintaining the appearance of source code. These conventions include file organization, indentation, comments, declarations the blanks, the opening and closing braces...

Software developers are encouraged to follow these conventions to help improve source code readability and ease software maintenance. The main reasons for the existence of these conventions are to reduce efforts when reading and understanding the source code and maintain the appropriate style standard for the programming language, in this way we focus on reviewing the most important points of our code. font rather than syntax and styles.
