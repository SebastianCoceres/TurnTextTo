TextFormatter is a simple tool for facilitate the writing of identifiers, variables and texts in html, css and js (and other languages too. Why not? )

Why another tool?
Yes, there are already several tools that perform these tasks, but having them all in the same tab and with just one click is a great plus

Just write what you want in the input and the code It will automatically take care of returning the formatted text in the nomenclature you want.
